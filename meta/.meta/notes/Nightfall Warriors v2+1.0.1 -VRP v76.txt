
DUO mode:

Internet connection required
Select Duo-mode in VR, a connection code will appear (stay on this screen)
Install and start Nightfall Warriors Companion's app from Google Play or Apple Store (it's free)
Insert the connection code that is shown in VR
Play!

Apple Store: https://apps.apple.com/us/app/nightfall-warriors/id6457343140
